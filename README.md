Welcome to Public Game Zone! This project serves as an interactive gaming portal that brings together various web games in one place. It is designed for gamers looking for quick fun right in the browser without the need for installation.

Key Features:

GAME LIBRARY: Quick access to several types of arcade and puzzle games.
INSTATNT PLAY: No downloads, everything runs instantly in HTML5.
USER EXPERIENCE: Intuitive controls and game interface optimized for a smooth experience.
DARK MODE DESIGN: Modern dark look that is easy on the eyes during long gaming sessions.

Technical development:

The project uses modern web technologies to ensure high performance (FPS) and stability:
Game canvas structure (Canvas).
Game animation and responsive UI.
Game logic, physics and game state management.
